# SocSci Review Hub

A browser-based interactive review and quiz web application 
built as a personal study tool for Social Science midterms 
preparation at the University of the Philippines Diliman.

## Features
- Interactive multiple-choice quiz system
- Progress tracking with score display
- Dashboard overview of study topics
- Color-coded answer feedback 
  (correct, wrong, reveal)
- Clean responsive UI with tab navigation
- No frameworks — pure HTML, CSS, and JavaScript

## Technologies Used
- **Languages:** HTML5, CSS3, JavaScript
- **Fonts:** Google Fonts (DM Serif Display, DM Sans)
- **Concepts:** DOM manipulation, dynamic UI rendering,
  quiz state management, responsive design

## How to Use
1. Download or clone the repository
2. Open `index.html` in any browser
3. Navigate using the top tabs
4. Start a quiz and track your score
